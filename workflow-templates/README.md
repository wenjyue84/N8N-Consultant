# N8N Workflow Templates

This directory contains reusable workflow templates for common automation scenarios.

## Available Templates

Templates will be added here as they are created. Each template includes:
- Workflow JSON file (importable into n8n)
- Documentation explaining the workflow
- Configuration examples

## Using Templates

1. Import the JSON file into your n8n instance
2. Configure the nodes with your credentials and parameters
3. Activate the workflow
